
• Sistema de Cadastro de Notas de Alunos

Este é um sistema para cadastro, consulta, atualização e exclusão de notas de alunos, desenvolvido em Python com Tkinter para a interface gráfica e PostgreSQL para o banco de dados.

• Requisitos

- Pycharm 3.X
- PostgreSQL

• Bibliotecas Necessárias

Para rodar o sistema, é necessário instalar as seguintes bibliotecas:

- psycopg2: Para comunicação com o banco de dados PostgreSQL.
- tkinter: Para a interface gráfica.
- faker: Para gerar dados fictícios (opcional, usado para popular a tabela).

• Como Executar

1. Instale as bibliotecas necessárias no seu ambiente Python.
2. Configure o banco de dados PostgreSQL no seu computador.
3. Execute o script Python para iniciar o sistema.
4. A interface gráfica será aberta, permitindo a inclusão, visualização, atualização e exclusão de alunos.

• Funcionalidades

- Adicionar Aluno: Permite adicionar um novo aluno com nome, matrícula e nota.
- Atualizar Aluno: Permite atualizar as informações de um aluno.
- Excluir Aluno: Permite excluir um aluno do banco de dados.
- Popular Aleatório: Adiciona 10 alunos fictícios ao banco.
